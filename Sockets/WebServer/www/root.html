You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /volume -- returns the volume of a cube or cylinder
  • /encrypt -- returns a firstname and a lastname encrypted

How to use the volume Get request

This request will calculate the volume of a cube or a cylinder. Each shape will have their own calculation and is based on what shape argument is passed.
Example cube run --> http://54.215.220.240:9000/volume?shape=cube&edgeLength=2
Example cylinder run --> http://54.215.220.240:9000/volume?shape=cylinder&radius=2&height=4

How to use the encrypt Get request

This request will display a firstname and lastname encrypted using BASE64.
Example run --> http://54.215.220.240:9000/encrypt?firstname=James&lastname=Lee
